HTML作成

スクリプトについて

  1. スクリプト名は、createHTML.jsです。comp_EC.jsで得られた検索結果であるJSONファイル、comp_EC_out_2019xxxx_xxxxxx.jsonを引数として与えます。ワイルドカードも使用できます。重複している項目は自動的に取り除かれます。
    ex1) node createHTML.js comp_EC_out_20190731_120000.json
ex2) node createHTML.js comp_EC_out_201907*.json
  2. createHTML.jsは[日付]_[0-9].htmlというファイルを生成します。200商品ごとに1つのHTMLを出力します。1つのファイルの多くの商品画像を詰めこむとスマホやタブレットで確認するときに表示が遅くなったりするためです。
  3. collect_ebay.jsの検索結果の商品情報、画像検索で見付けた類似画像の商品情報を、一商品ごとにHTMLとして表示します。主な目的は、本当に同一商品なのか、価格差はあるか、を目視で容易に確認するためです。

出力されるHTML

  • MacやWindowsであればブラウザに生成されたHTMLをドラッグ＆ドロップすれば見ることができます。その他の環境での方法は調べて下さい。
  • まず始めに商品画像だけの一覧が出てきます。その画像をクリックすると、その商品情報、及び類似画像検索結果へ移動します。
  • COPYボタンを押すと、商品名、URL、価格がクリップボードにコピーされます。その際、各項目の区切り文字をカンマやスペース、タブなどにしておくこともできますので、ご相談下さい。例えばタブで区切っておくとGoogleスプレッドシートにペーストすれば、各項目を別々のセルに簡単に貼り付けることができます。
