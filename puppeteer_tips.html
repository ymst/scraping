puppeteer TIPs

puppeteerの自分用メモ

puppeteerを使っていろいろスクレイピングを始めてから、検索して得た情報を以下にメモしていく。

ブラウザを開く

まず、ユーザエージェントの設定について。

page.setUserAgent([UA 文字列]);
page.setUserAgent('Mozilla/5.0 ..'); //特に指定しない場合

プロキシを使う設定について。

puppeteer.launch({
  args: [
    '--lang=ja,en-US,en',
    '--no-sandbox',
    '--proxy-server
      = [IPアドレス]:[ポート]'
  ]
});

高速化のため、画像や動画を読み込まない設定をする。

page.on('request', interceptedRequest => {
  if (
    interceptedRequest.url().endsWith('.png') ||
    interceptedRequest.url().endsWith('.jpg') ||
    interceptedRequest.url().endsWith('.mov') ||
    interceptedRequest.url().endsWith('.mp4'))
      interceptedRequest.abort();
    else
      interceptedRequest.continue();
});

タブの取得、タブの作成について。

// 1枚めのタブを取得する
var page = (browser.pages())[0];
// 新たなタブを開く
var page = browser.newPage();

URLを開く

普通に開くには。

page.goto([URL]);

少し細かな設定。タイムアウトや読み込み終了条件など。

page.goto([URL],{
    timeout: [msec],
    waitUntil: [
      "load",
      "domcontentloaded",
      "networkidle0"
    ]
});

page.gotoが失敗したことを検出する。

try {
    page.goto([URL]);
}
catch(e) {
    console.log('ページを開けませんでした');
}

page.gotoの戻り値でHTTP statusコードなどを取得できる。

const RESULT = await page.goto([URL]);
console.log(RESULT._status);

クリック動作

調べた限りでは2通りの方法が見つかった。page.click()の方では動作しない場合が結構あった。

// クリックする要素を取得
const Atag = await page.$('a[id="sample"]');
// 上で取得した要素をクリック
await Atag.click();

もうひとつの方法はこちら。

// クリックする要素を取得
await page.evaluate(() => {
  const A = document.querySelector
    ('a[id="sample"]');
  A.click();
});

上のどちらの方法を使う場合でも、クリックする要素がなければエラーで止まるので、クリックする要素があるかどうかチェックしたいとき

try {
  errFlag = false;
  await page.waitFor('a[id="sample"]');
}
catch(e) {
  errFlag = true;
}
if (errFlag) console.log('Atag not found.');

セレクト動作

ドロップダウンリストから項目を選択する方法。

await page.select('select[id="sample"]', [値]);

それでも要素を選択できない場合はこちら。

await page.evaluate(() => {
  let elem = 
    document.querySelector('select[id="test"]');
  elem.querySelector('option[value="n"]').selected = true;
  let event = 
    new Event('change', { bubbles: true });
  event.simulated = true;
  elem.dispatchEvent(event);
});

要素を取り出す

色んな要素を取り出す方法

outerHTMLで取り出してreplaceで切り貼りする方法が便利かな


// AタグからHREFを取り出す
const HREF =
  await(
    await(
      await page.$('a[id="sample"]')
    ).getProperty('href')
  ).jsonValue();
// 指定した要素を含むHTMLを取り出す
const HTML =
  await(
    await(
      await page.$('a[id="sample"]')
    ).getProperty('outerHTML')
  ).jsonValue();
// 指定した要素の中のテキストだけを取り出す
const HTML =
  await(
    await(
      await page.$('a[id="sample"]')
    ).getProperty('textContent')
  ).jsonValue();
