puppeteer TIPs

puppeteerの自分用メモ

puppeteerを使っていろいろスクレイピングを始めてから、検索して得た情報を以下にメモしていく。

ブラウザを開く

まず、ユーザエージェントの設定について。


page.setUserAgent([UA 文字列]);
page.setUserAgent('Mozilla/5.0 ..'); //特に指定しない場合

プロキシを使う設定について。


puppeteer.launch({
    args: ['--lang=ja,en-US,en', '--no-sandbox', '--proxy-server=[IPアドレス]:[ポート]']
    });

高速化のため、画像や動画を読み込まない設定をする。


page.on('request', interceptedRequest => {
    if (interceptedRequest.url().endsWith('.png') || interceptedRequest.url().endsWith('.jpg') || interceptedRequest.url().endsWith('.mov') || interceptedRequest.url().endsWith('.mp4'))
        interceptedRequest.abort();
    else
        interceptedRequest.continue();
});

タブの取得、タブの作成について。


// 1枚めのタブを取得する
var page = (browser.pages())[0];
// 新たなタブを開く
var page = browser.newPage();

URLを開く

普通に開くには。


page.goto([URL]);

少し細かな設定。タイムアウトや読み込み終了条件など。


page.goto([URL],{
    timeout: [msec],
    waitUntil: ["load", "domcontentloaded", "networkidle0"]
    }
);

page.gotoが失敗したことを検出する。


try {
    page.goto([URL]);
}
catch(e) {
    console.log('ページを開けませんでした');
}

page.gotoの戻り値でHTTP statusコードなどを取得できる。


const RESULT = await page.goto([URL]);
console.log(RESULT._status);
